# Task 2 — Shared Essay Framework

Task 2 is an **argumentative/discursive essay** in academic or semi-formal style.

## Universal rules

- Minimum **250 words**; penalty below 250.
- **Must include a conclusion** (unlike Task 1).
- Clear paragraphing: introduction → body paragraph(s) → conclusion.
- Four criteria × 25% (see scoring_matrix.md); Task 2 counts **double** toward total Writing score.

## Standard essay structure

1. **Introduction** — Paraphrase topic + thesis/clear stance (when required).
2. **Body 1** — One central idea, explained and supported with example/reasoning.
3. **Body 2** — Second central idea (and Body 3 if needed for two-part questions).
4. **Conclusion** — Summarise position; no new major ideas.

## Mistake categories (Task 2)

Grammar, Spelling, Vocabulary, Word choice, Sentence structure, Awkward phrasing, Cohesion, Punctuation, Task response, Other.

## Essay type detection

The prompt determines subtype — classify before scoring:

| Subtype | Typical prompt signals |
|---------|------------------------|
| **Opinion** | "To what extent do you agree or disagree?" |
| **Discussion** | "Discuss both views and give your opinion" |
| **Advantages/disadvantages** | "Do the advantages outweigh the disadvantages?" |
| **Problem/solution** | "What are the causes…? What solutions…?" |
| **Two-part** | Two distinct questions in one prompt |

Each subtype has different Task Response expectations — use the matching type-specific guide.
