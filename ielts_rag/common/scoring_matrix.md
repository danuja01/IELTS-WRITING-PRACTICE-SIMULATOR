# IELTS Writing — Official Scoring Matrix

All Task 1 and Task 2 responses are marked on **four criteria**, each worth **25%** of the task score.

| Criterion | Task 1 | Task 2 |
|-----------|--------|--------|
| Primary task criterion | **Task Achievement (TA)** | **Task Response (TR)** |
| Organisation | Coherence & Cohesion (CC) | Coherence & Cohesion (CC) |
| Vocabulary | Lexical Resource (LR) | Lexical Resource (LR) |
| Grammar | Grammatical Range & Accuracy (GRA) | Grammatical Range & Accuracy (GRA) |

**Overall band** = average of the four criteria, rounded to the nearest **0.5** (e.g. 6.25 → 6.5).

---

## Task Achievement / Task Response — Band descriptors (summary)

| Band | Task 1 (TA) | Task 2 (TR) |
|------|-------------|-------------|
| 9 | Fully satisfies all requirements; clear overview; key features skilfully selected | Fully addresses all parts; fully developed position with relevant, extended ideas |
| 8 | Covers requirements; overview clear; key features skilfully selected | Sufficiently addresses all parts; well-developed ideas with relevant support |
| 7 | Covers requirements; clear overview of main trends/features; key features highlighted | Addresses all parts; clear position; main ideas extended and supported |
| 6 | Addresses requirements; overview attempted; key features selected though details may be irrelevant | Addresses all parts though some more fully than others; relevant position though conclusions may be unclear |
| 5 | Generally addresses task; overview missing/unclear; key features not adequately covered | Addresses task only partially; position unclear; ideas limited/undeveloped |

---

## Coherence & Cohesion — Band descriptors (summary)

| Band | Expectation |
|------|-------------|
| 7+ | Logically organised; clear progression; range of cohesive devices used appropriately |
| 6 | Arranged coherently; overall progression clear; cohesive devices used (may be mechanical) |
| 5 | Organisation evident but not wholly logical; limited cohesive devices |

---

## Lexical Resource — Band descriptors (summary)

| Band | Expectation |
|------|-------------|
| 7+ | Flexible use of vocabulary; less common items; occasional inaccuracies acceptable |
| 6 | Adequate range; some less common items; errors do not impede communication |
| 5 | Limited range; frequent errors in word choice/spelling may cause difficulty |

---

## Grammatical Range & Accuracy — Band descriptors (summary)

| Band | Expectation |
|------|-------------|
| 7+ | Variety of complex structures; frequent error-free sentences; few errors |
| 6 | Mix of simple and complex; some errors but meaning generally clear |
| 5 | Limited range; frequent errors; may cause difficulty for reader |

---

## Scoring workflow for AI examiners

1. **Classify** the question subtype (visual type or essay type).
2. **Apply subtype-specific TA/TR expectations** (trends vs comparisons vs map changes, etc.).
3. **Assign a band for each criterion** using the descriptors above.
4. **List mistakes separately** — errors inform feedback but do not automatically cap bands when communication remains clear.
5. **Compute overall band** from the four criterion scores.
