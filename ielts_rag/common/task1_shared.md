# Task 1 — Shared Report Framework

Task 1 is an **information-transfer report**, not an essay.

## Universal rules

- Minimum **150 words**; penalty below 150.
- **No opinion**. **No conclusion paragraph**.
- Formal, objective tone.
- Four criteria × 25% (see scoring_matrix.md).

## Required 4-paragraph structure

1. **Introduction** — Paraphrase the task prompt (what is shown, units, time period). No data.
2. **Overview** — Main trends, differences, or changes at a glance. **No specific figures.**
3. **Body 1** — Key features group 1 with selective data/comparisons.
4. **Body 2** — Key features group 2 with selective data/comparisons.

An absent or vague overview typically caps Task Achievement at **Band 5–6**.

## Dynamic vs static data visuals

| Type | Time axis? | Language focus |
|------|------------|----------------|
| **Dynamic** (line graph, time-series bar/table) | Yes | Trends: rose, fell, peaked, fluctuated |
| **Static** (single-year bar, pie, static table) | No | Comparisons: highest, lowest, twice as much, accounted for |

Maps and process diagrams follow their own frameworks (see type-specific guides).

## Mistake categories (Task 1)

Grammar, Spelling, Vocabulary, Word choice, Sentence structure, Awkward phrasing, Cohesion, Punctuation, Task achievement, Data accuracy, Overview, Other.

## Calibration note

A clear 4-paragraph report with overview + key features + minor language errors typically scores **6.0–6.5 overall**, not 5.5.
