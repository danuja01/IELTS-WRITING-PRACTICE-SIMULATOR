# Discussion Essay — Type-Specific Evaluation Guide

## Prompt identification

Signals: "Discuss both views and give your opinion", "Some people believe X, while others think Y. Discuss both views."

## Task Response priorities

| Priority | What to assess |
|----------|----------------|
| **High** | Both views presented fairly; writer's opinion stated clearly |
| **Medium** | Balanced development of each view before opinion |
| **Low** | Only one view discussed; opinion missing or buried |

## Required structure

1. **Introduction** — Paraphrase both views + outline that essay will discuss both and state opinion
2. **Body 1** — View A (reasons/examples) — can use neutral tone
3. **Body 2** — View B (reasons/examples)
4. **Conclusion** — **Your opinion** clearly stated with brief justification

## Opinion placement

Opinion may appear in introduction, conclusion, or both — but **must be explicit** somewhere.

## Common TR weaknesses

- One-sided essay (only own view)
- Both views listed superficially without development
- No personal opinion in conclusion
- Opinion in body 1 only, then forgotten

## Lexical/cohesion focus

View A: Proponents argue, Supporters claim, On one hand
View B: Conversely, However, On the other hand
Opinion: In my opinion, I believe, While both views have merit…

## Model essay tone

Balanced, then decisive in conclusion. Target **280+ words**.
