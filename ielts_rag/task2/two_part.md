# Two-Part Question — Type-Specific Evaluation Guide

## Prompt identification

Two distinct questions in one prompt, e.g.:
- "Why is this happening? Is it a positive or negative development?"
- "What are the reasons? Do you think this trend will continue?"

## Task Response priorities

| Priority | What to assess |
|----------|----------------|
| **High** | **Both questions answered** with adequate development |
| **Medium** | Equal or proportional coverage of each part |
| **Low** | One question ignored or answered in one sentence |

## Required structure

1. **Introduction** — Paraphrase topic + preview both answers
2. **Body 1** — Answer to **Question 1** (fully developed)
3. **Body 2** — Answer to **Question 2** (fully developed)
4. **Conclusion** — Summarise both answers

## Opinion on "positive or negative"

When the second part asks "Is it a positive or negative development?", a **clear one-sided opinion** is valid. The student does NOT need a separate disadvantages paragraph or equal coverage of both sides — only a justified stance.

## Common TR weaknesses

- Strong answer to Q1, token answer to Q2
- Merging both questions into one undifferentiated body
- Conclusion only addresses one question
- Misidentifying which part is being asked
- **False flag**: penalising a clear positive/negative opinion for not listing both sides

## Lexical/cohesion focus

Signal answers clearly: "Regarding the first question…", "With respect to whether…"
Use separate paragraph topics — do not blend unrelated answers.

## Model essay tone

Direct, structured, each part clearly signposted. Target **280+ words**.
