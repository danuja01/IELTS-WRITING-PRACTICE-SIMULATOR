# Problem / Solution — Type-Specific Evaluation Guide

## Prompt identification

Signals: "What are the causes?", "What problems does this cause?", "What solutions can you suggest?", "How can this issue be solved?"

## Task Response priorities

| Priority | What to assess |
|----------|----------------|
| **High** | Both problem(s) AND solution(s) fully addressed |
| **Medium** | Solutions are practical and linked to stated problems |
| **Low** | Causes essay with one sentence on solutions; or solutions unrelated to problems |

## Required structure

1. **Introduction** — Paraphrase issue + outline that causes/problems and solutions will be discussed
2. **Body 1** — Causes or problems (developed with examples)
3. **Body 2** — Solutions (specific, actionable, linked to Body 1)
4. **Conclusion** — Summarise problem + best solutions

## Two-part cause/solution prompts

Both parts must receive **roughly equal development**. Do not write 80% on causes.

## Common TR weaknesses

- Vague solutions ("the government should do something")
- Solutions not matching identified problems
- Only describing problems without solutions
- Suggesting solutions for problems never mentioned

## Lexical/cohesion focus

Problem: stem from, result in, give rise to, exacerbate, widespread issue
Solution: implement, address, mitigate, tackle, effective measure would be
Linking: Consequently, To combat this, One way to address this

## Model essay tone

Problem-focused then solution-oriented, practical. Target **280+ words**.
