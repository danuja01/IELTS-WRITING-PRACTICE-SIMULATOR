# Mixed Charts — Type-Specific Evaluation Guide

## Visual identification

**Two different chart types** in one task (e.g. line + bar, pie + table, bar + pie). Student must integrate both visuals.

## Task Achievement priorities

| Priority | What to assess |
|----------|----------------|
| **High** | Both visuals addressed; balanced coverage; overview mentions both |
| **Medium** | Relationship or contrast between the two visuals |
| **Low** | 80/20 split ignoring one chart |

## Overview expectations

Must reference **both visuals** (e.g. "While the line graph shows growth over time, the bar chart compares categories in 2020."). Integrated summary, not two separate overviews.

## Body paragraph grouping

- **Visual 1 key features** in one body, **Visual 2** in the other, OR
- **Theme-based**: compare what both show about same topic
- Explicitly link visuals where data relates

## Language focus

Combine trend vocabulary (if line/time-series present) with comparison/proportion language as appropriate per visual. Clearly signal which visual is being discussed.

## Common weaknesses

- Overview only describes one chart
- Ignoring smaller visual entirely
- Treating as two separate mini-reports without integration
- Wrong chart-type language (trends on static pie)

## Data accuracy checks

Figures must match the correct visual. Attributing bar data to line graph = **Data accuracy** mistake.

## Model report tone

Integrated, balanced. Target **170–210 words**.
