# Line Graph — Type-Specific Evaluation Guide

## Visual identification

A line graph shows **continuous change over time**. Multiple lines = multiple categories compared across the same timeline.

## Task Achievement priorities

| Priority | What to assess |
|----------|----------------|
| **High** | Overall trends per line; meaningful line-to-line comparisons; clear overview of dominant patterns |
| **Medium** | Peaks, troughs, crossover points, periods of stability |
| **Low** | Every year/point individually; minor fluctuations |

## Overview expectations

Summarise **overall trajectories** (e.g. "X rose steadily while Y declined throughout the period"). No figures in overview.

## Body paragraph grouping

- Group by **similar trend direction** (all rising / all falling), OR
- Group by **time period** (early vs late), OR
- Compare **two related lines** in one body, remaining lines in the other.

## Tense guidance

| Section | Tense |
|---------|-------|
| Introduction / overview | Present simple ("The graph shows…") |
| Past data | Past simple ("rose to", "fell from") |
| Spanning to present | Present perfect where appropriate |

## Lexical resource — trend vocabulary

increased, decreased, rose, fell, peaked at, bottomed out at, fluctuated, remained stable, experienced a gradual rise, witnessed a sharp decline, plateaued, levelled off

## Common weaknesses (penalise in TA/LR)

- Describing every year without comparison
- Listing figures without trend language
- Missing the main trend or overview
- Using static comparison language ("was higher than") without trend verbs

## Data accuracy checks

Verify figures against the chart. Wrong start/end values or reversed trends = **Data accuracy** mistakes.

## Model report tone

Analytical, comparative, trend-focused. Target **160–190 words**.
